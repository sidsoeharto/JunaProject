# JUNA PROJECT

## BACKEND INSTALLATION

- run 'cd JunaBackend' on the root folder
- run 'npm install'
- run 'npm start'

## FRONTEND INSTALLATION

- run 'cd JunaFrontend' on the root folder
- run 'npm install'
- run 'npm run android'
